# 逆变换方法生成随机变量

## 一、概述

逆变换方法是一种从特定概率分布生成随机变量的基本技术。该方法通过将均匀随机变量转换为目标分布的随机变量来实现抽样。

逆变换方法的核心步骤：

1. 先生成最简单的：$U\sim U(0,1)$ 均匀随机数
2. 求目标分布累积分布函数 $F(x)$ 的反函数 $F^{-1}(u)$
3. 令 $X=F^{-1}(U)$，得到的 X 就服从你想要的目标分布

**适用条件**：目标分布的累积分布函数（Cumulative Distribution Function, CDF）必须存在解析的逆函数。

**局限性**：对于CDF不可逆的分布（如正态分布），需要使用其他方法如接受拒绝法。

**例子**：骰子取值：\(1,2,3,4,5,6\)，每个概率\(\dfrac16\)累积分布：
\(F(x)=
\begin{cases}
0, &x<1\\
\dfrac16, &1\le x<2\\
\dfrac26, &1\le x<3\\
\cdots\\
1, &x\ge6
\end{cases}\)
1. 生成 \(U\in(0,1)\)
2. 判断落在哪个区间，直接得到对应点数这就是离散型逆变换

**目的**：
逆变换法就是将计算机易生成的 0 到 1 均匀随机数，转化为符合现实场景特定概率分布的随机变量，以此得到符合真实规律的随机样本，用来开展各类动态流程仿真与实际场景推演。

> 像是统计量的计算（均值、概率、稳态指标）这些静态量不需要生成随机变量。
> 当需要模拟、仿真，就需要生成一系列随机变量（就是生成样本），样本本身符合分布，有了这些样本后可以再增加条件（实际限制）来模拟实际情况。
> 比如已知顾客是成指数分布来到店里的（前一个客人和后一个客人是没有独立事件），那就可以生成一些列客人样本（用客人到店时间表示）。在这种假设下，增加条件例如用户等待超过多少时间会离开，来进一步模拟店里的供应能力以及需要配置的服务人员。 如果前一个客人与后一个客人就是有关联，那么他就不符合指数分布，那就不能使用这种分布来生成随机变量。
> 逆变换的作用就是: 已知符合某种分布, 将均匀分布通过逆变化生成符合分布的随机变量。
> 过程: 累计分布函数CDF求逆及逆变换得到符合分布随机变量

## 二、基础概念

### 2.1 概率质量函数（Probability Mass Function, PMF）

**定义**：将样本空间中的元素映射到对应概率的函数。$p(x)=P(X=x)$

**性质**：
- 所有概率值在[0, 1]范围内
- 样本空间所有概率之和为1

**示例**：离散均匀分布（掷骰子）
$$P(a) = \frac{1}{6}, \quad a \in \{1, 2, 3, 4, 5, 6\}$$

### 2.2 累积分布函数（CDF）

**定义**：概率质量从最小值累加到$x$的总概率。$F(x)=P(X \le x)$

**性质**：
- 输入趋近负无穷时，CDF值为0
- 输入趋近正无穷时，CDF值为1
- 范围始终为[0, 1]

**离散情况计算**：
$$F(x) = \sum_{k \leq x} P(k)$$

**示例**：掷骰子的CDF
- $F(x) = 0, \quad x < 1$
- $F(x) = \frac{1}{6}, \quad 1 \leq x < 2$
- $F(x) = \frac{2}{6}, \quad 2 \leq x < 3$
- ...以此类推直至 $F(x) = 1$

## 三、逆变换方法

### 3.1 核心思想

利用CDF的范围特性（[0, 1]），通过以下步骤生成随机变量：
1. 生成一个均匀随机变量 $U \in [0, 1]$
2. 将 $U$ 输入逆CDF函数
3. 得到目标分布的抽样结果

### 3.2 逆变换过程

$$
PMF \xrightarrow{\text{求和}} CDF \xrightarrow{\text{逆变换}} \text{生成随机数}
$$

### 3.2 算法步骤

**连续情况**：
1. 求目标分布的CDF：$F(x)$
2. 求CDF的逆函数：$G(u) = F^{-1}(u)$
3. 从标准均匀分布抽取：$U \sim \text{Uniform}(0, 1)$
4. 计算输出：$X = G(U)$

**离散情况**（需要累积循环）：
1. 计算累积概率
2. 抽取均匀随机数 $U$
3. 累积概率质量直到超过 $U$
4. 返回对应的结果值

### 3.3 工作原理

由于CDF将样本空间映射到[0, 1]，其逆函数将[0, 1]映射回样本空间。均匀分布在[0, 1]上每个数值等概率，通过逆CDF映射后，各结果的概率与原分布一致。


## 四、伪随机数生成

### 4.1 概念

伪随机数生成器（Pseudo Random Number Generator, PRNG）是确定性算法，用于产生看似随机的数字序列。

**特点**：
- 确定性：已知种子可完全预测输出
- 周期性：序列最终会重复
- 可复现性：相同种子产生相同序列

### 4.2 线性同余生成器（Linear Congruential Generator, LCG）

**公式**：
$$X_{n+1} = (aX_n + c) \mod M$$

**归一化**：输出 $X_{n+1}/M$ 得到[0, 1]范围内的值。

**参数选择**：a、c、M的选择影响生成器质量，通常M取大数。

**Python实现**：
```python
import numpy as np
U = np.random.uniform(0, 1)  # 直接使用
```

## 五、Python实现示例

### 5.1 离散均匀分布（掷骰子）

```python
import numpy as np

# 定义样本空间和概率
outcomes = [1, 2, 3, 4, 5, 6]
probabilities = [1/6] * 6

# 计算CDF
cdf = np.cumsum(probabilities)  # [1/6, 2/6, 3/6, 4/6, 5/6, 1]

# 生成样本
n_samples = 10000
U = np.random.uniform(0, 1, n_samples)  # 均匀分布生成随机数
samples = [outcomes[np.searchsorted(cdf, u)] for u in U] # searchsorted二分法找到u的 idx
```

### 5.2 一般离散分布（循环累积法）

```python
def inverse_transform_discrete(pmf_func):
    U = np.random.uniform(0, 1)
    F = 0  # 累积概率
    a = 1  # 当前结果，a = {1,2,3,4,5,6}
    
    while U > F:
        F += pmf_func(a)
        a += 1
    
    return a - 1
```

pmf概率质量函数
区别[5.1]：
- 5.1 使用二分法，样本空间有限
- 5.2 使用累加法，样本空间无限

### 5.3 连续分布（指数分布）

**CDF推导**：
- PDF（Probability Density Function，概率密度函数）：$f(x) = \lambda e^{-\lambda x}$
- CDF：$F(x) = 1 - e^{-\lambda x}$

**求逆CDF**：
$$u = 1 - e^{-\lambda x} \implies x = -\frac{\ln(1-u)}{\lambda}$$

**Python实现**：
```python
def inverse_cdf_exponential(u, lambda_param):
    return -np.log(1 - u) / lambda_param

# 生成样本
n_samples = 10000
U = np.random.uniform(0, 1, n_samples)
samples = inverse_cdf_exponential(U, lambda_param=1)
```
